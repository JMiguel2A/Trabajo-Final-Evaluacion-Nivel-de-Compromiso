# Trabajo Final Evaluacion Nivel de Compromiso

Este programa fue desarrollado en Python como trabajo final de Fundamentos de Programación.

El sistema analiza sesiones de usuarios utilizando una matriz que almacena:

- ID del usuario
- Tiempo de uso en segundos
- Número de clicks realizados

Mediante una función, el programa clasifica el nivel de compromiso de cada usuario en:

- Alto
- Medio
- Bajo
